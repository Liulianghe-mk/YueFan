约饭 · 餐饮社交全栈项目 | 微信小程序 / Spring Boot / Vue 3 | 独立开发
三端架构：微信小程序 + Spring Boot 后端 + Vue 3 管理后台，覆盖约饭社交完整业务链路
后端：Spring Boot 3.2、JPA、Flyway 数据库迁移，双 JWT 鉴权，20+ RESTful API
小程序：15+ 页面，实现约饭报名、动态社区、私信、用户关注等核心功能
管理端：Vue 3 + Element Plus 实现活动、内容、用户运营管理
